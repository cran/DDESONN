Plot Interface — Scenario 1

This vignette demonstrates the Scenario 1 plot configuration interface in DDESONN.

Independent Call Sites (training_overrides)

Scenario 1 configures plotting through three independent call sites inside training_overrides:

  1. training_overrides$per_epoch_plots
  2. training_overrides$performance_relevance
  3. training_overrides$evaluate_predictions_report_plots

Each plot category is defined and controlled separately.

This represents the minimal-integration bridge interface, where plot configuration remains distributed across training components.
